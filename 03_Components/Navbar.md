## Options
### Floating
To make it floating on top, just add nav.allways-on-top
``` html
<nav class="always-on-top">
	/* banner, navbar, toolbar */
</nav>
```

### Island
If you want your navabr to be an island, just add div.navbar.island
``` html
<div class="navbar island">
	/* navbar */
</div>
```
