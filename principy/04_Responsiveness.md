A) 12 grid system
	- Width:  .w-1 .w-2 ... .w-12
	- Mobile Width: .m-w-1 ... m-w-12
B) Pixels
	- Width: .w-50-px .w-100-px  ... .w.- TODO: DOPLN
	- Width mobile .m-w-50-px ...
